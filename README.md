# WAD-Tool-For-Hotline-Miami-1
This tool allows you to extract and pack WAD archive file for Hotline Miami 1

---

To extract just select the HotlineMiami_GL.wad file and choose Extract WAD in the tool
You’ll be able to see and edit any file inside the archive in this folder extracted_files
Once you’re done editing select PACK WAD to rebuild the archive
and the tool will automatically create a new file called modified.wad
